# Privacy Policy - ENGAGEDLOOP

**Effective Date:** April 23, 2026

## Data Collection

**ENGAGEDLOOP does NOT collect, store, or transmit any personal information or user data.**

### What we access:
- **Tweet text content** - Only when you explicitly click the "Reply Assist" button on a specific tweet
- **User's Gemini API key** - Stored locally in your browser's extension storage (chrome.storage.local)

### What we do NOT access:
- Your Twitter/X login credentials or session tokens
- Your DMs (Direct Messages)
- Your follower lists
- Your account information
- Any browsing history outside of x.com
- Cookies or authentication tokens

## Data Usage

### Tweet Text Processing
- Tweet text is read **only** when you click the "Reply Assist" button
- The text is sent to **Google Gemini API** to generate reply suggestions
- Tweet text is **not stored** permanently - only temporarily held in memory during the generation process
- We do not build databases of tweets or user content

### API Key Storage
- Your Gemini API key is stored **locally** in your browser
- The key is **never** transmitted to our servers (we don't have servers)
- The key is used only to authenticate requests to Google's Gemini API

## Data Sharing

**We do not share any data with third parties.**

The only external service used is:
- **Google Gemini API** - Receives tweet text only when you explicitly request reply generation

## User Control

You have full control over your data:
- You can delete your API key at any time via the extension popup
- You can uninstall the extension to remove all stored data
- No data persists after extension removal

## Compliance

This extension complies with:
- Chrome Web Store Developer Program Policies
- Twitter/X Terms of Service (no automation, explicit user action required)
- GDPR principles (data minimization, user control)

## Contact

For privacy concerns, please contact the developer through the Chrome Web Store listing.

**TL;DR:** We only read tweet text when YOU click a button, and we don't store anything. Your API key stays on your computer.
