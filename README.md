# // ENGAGEDLOOP // [STATUS: LIVE]

Brutalist engineering for smarter X (Twitter) interactions. 
A high-performance Chrome Extension + Web Dashboard sync system.

## 🛠️ ARCHITECTURE
- **WEB DASHBOARD**: Static HTML/JS deployed on Vercel.
- **CHROME EXTENSION**: manifest v3 with content-script injection.
- **DATABASE**: Supabase (Auth + Usage Tracking).
- **AI ENGINE**: Gemini 2.5 Flash.

## 📂 STRUCTURE
- `/extension` - The core Chrome Extension source code.
- `/enagedloop.items` - Assets & screenshots for Chrome Web Store.
- `index.html` - Main landing page.
- `dashboard.html` - User profile & usage metrics.
- `login.html` - Supabase Auth gateway.
- `vercel.json` - Static routing configuration.

## ✨ FEATURES
- ✅ **Reply Assist** - Generate 3 natural, human-like replies to any tweet.
- ✅ **Post Mode** - Generate founder-style X posts from your ideas.
- ✅ **Personal Style** - AI mimics your writing style from samples.
- ✅ **Supabase Sync** - Auth, usage limits, and cross-device sync.
- ✅ **Brutalist UI** - Raw, high-contrast engineering aesthetic.

## 🚀 DEPLOYMENT
1. **WEB**: Push to GitHub -> Vercel auto-deploys.
2. **EXTENSION**: Run `deploy-extension.bat` -> Upload `.zip` to Chrome Dev Console.
